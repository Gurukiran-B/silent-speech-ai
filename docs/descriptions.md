# Silent Speech AI - Project Descriptions

## GitHub Repository Description
A real-time proof-of-concept system that decodes silent speech from webcam video using MediaPipe facial landmarks and a K-Nearest Neighbors (KNN) model.

## Resume Description (100 words)
Engineered a real-time Silent Speech AI system designed to decode spoken words solely from visual lip movements. Developed a custom dataset and an end-to-end machine learning pipeline using Python, OpenCV, and MediaPipe to extract key facial landmarks. Engineered 7 critical mouth-shape features to train a K-Nearest Neighbors (KNN) classifier capable of recognizing a 10-word vocabulary with ~50% baseline accuracy. Built a client-server architecture featuring a responsive frontend for video capture and a Flask backend for real-time inference. This proof-of-concept lays the groundwork for advanced temporal modeling approaches to assist individuals with speech impairments.

## LinkedIn Project Description
🚀 I recently built a Version 1 proof-of-concept for a Silent Speech AI!

This project explores the fascinating challenge of decoding spoken words solely from silent lip movements. Using Computer Vision and Machine Learning, the system captures real-time video, extracts facial landmarks, and predicts words without any audio input. 

Key technical highlights:
- 👁️ **Computer Vision:** Used MediaPipe and OpenCV to accurately track lip movements in real time.
- ⚙️ **Feature Engineering:** Extracted mathematical features like Mouth Aspect Ratio and Mouth Area from facial landmarks.
- 🤖 **Machine Learning:** Trained a K-Nearest Neighbors (KNN) model on a custom 10-word dataset as an initial baseline (~50% accuracy).
- 🌐 **Full-Stack:** Connected the ML pipeline to a responsive web interface using a Flask backend.

While this is an early prototype (limited vocabulary and no temporal modeling yet), it was a fantastic dive into the intersection of CV and ML. Excited to explore more advanced techniques like LSTMs or Transformers for this problem in the future!
